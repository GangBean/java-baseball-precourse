#**************[프리코스 1주차 숫자야구게임 기능 리스트]**************

##UI 기능
1. 숫자 입력 안내
   1. '숫자를 입력해주세요 : '
   2. 숫자 입력
2. 판정결과 출력
   1. '1볼 1스트라이크', '낫싱'
3. 게임 종료 여부 입력 안내
   1. '게임을 새로 시작하려면 1, 종료하려면 2를 입력하세요.'
   2. 게임 종료 여부 입력
   3. 미정의값 입력시 반복 입력 요구 : 1,2 외의 값 입력시 재입력요구

##도메인 기능
1. 정답 생성
   1. 랜덤수 생성
   2. 정답 저장 
2. 볼 생성
   1. 입력값 검증
      1. 숫자 검증 : 1 ~ 9
      2. 갯수(인덱스) 검증 : 3개(0 ~ 2)
3. 숫자 판별
   3. validation
      1. 자릿수체크
      2. 숫자 여부 체크
      3. IllegalArgumentException
   4. 스트라이크 판정
   5. 볼 판정
4. 판별 결과 생성
   1. 낫싱 콜 생성
   2. 볼 콜 생성
   3. 스트라이크 콜 생성
   4. 최종 콜 조합 생성
